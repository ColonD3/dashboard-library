# Dashboard Playground

Tiny static dashboard playground.

## Folder structure

```txt
dashboard-library/
  index.html
  style.css
  dashboards/
    drive-test-remix/
      index.html
  assets/
    images/
```

## Add a new dashboard

1. Make a folder inside `dashboards/`
2. Put an `index.html` in that folder
3. Add a card/link to the homepage `index.html`

Example dashboard URL after GitHub Pages is enabled:

```txt
https://ColonD3.github.io/dashboard-library/dashboards/drive-test-remix/
```
